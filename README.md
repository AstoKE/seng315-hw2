1. Node.js 18+ must be installed:
- node -v

2. Clone the repo:
git clone <repo-url>
cd seng315-hw2 (or whatever your repo name is)

3. Install dependencies:
npm install

4. Create the .env file:
Copy the .env.example file:
- Windows: copy .env.example .env
- Mac/Linux: cp .env.example .env

5. Create the data folder:
mkdir data

6. Run the application:
npm run dev

7. Go to the admin/debug UI in the browser:
http://localhost:3000/

- In the Users section, create a user / list
- In the Messages section, create a message / list
